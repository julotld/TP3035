# projet TP-3035
## Objectif du projet
Ce projet a pour objectif d'analyser et de modéliser les coûts médicaux en assurance santé.  
Il suit les étapes classiques d'un processus actuariel : ETL, analyse exploratoire, modélisation et comparaison des modèles.

J’ai choisi d’utiliser un jeu de données médicales sur les coûts des soins de santé car ces données permettent d’illustrer concrètement les problématiques actuarielle en assurance santé. Elles contiennent des variables individuelles (âge, IMC, habitudes de vie, nombre d’enfants) qui influencent directement les coûts médicaux, ce qui rend possible la construction de modèles de tarification et d’évaluation des risques réalistes. Ce jeu de données est suffisamment riche pour appliquer un processus ETL, réaliser une analyse exploratoire et construire différents modèles actuariels tout en restant simple à manipuler pour un projet académique. En effet, on a un nombre assez important d'observation pour faire une analyse de base correcte, ainsi que des variables pertinente a transformer, analyser et modéliser.
